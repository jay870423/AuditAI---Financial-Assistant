## 1. 快速开始

### 访问系统
通过浏览器访问 AuditAI 应用链接（建议使用 Chrome、Edge 或 Safari 浏览器以获得最佳体验）。应用支持 PC 端和移动端访问。

### 登录账户
为了使用 AI 分析功能并确保数据安全，您需要先登录。
1. 点击右上角或侧边栏底部的 **“登录” (Sign In)** 按钮。
2. 选择您偏好的登录方式：
   - **微信登录 (WeChat)**：适合中国用户，快捷方便。
   - **Google / GitHub**：适合国际用户或开发者。
3. 登录成功后，即可解锁所有高级分析功能。

## 2. 功能模块详解

### 2.1 财务数据分析

此模块用于分析银行流水、日记账或交易记录，快速发现异常。

**操作步骤：**

1. **进入模块**：点击仪表盘的 **“数据分析”** 卡片或左侧导航栏的 **“数据分析”**。
2. **准备数据**（三种方式）：
   - **方式 A：使用 Excel 模板（推荐）**  
     点击右上角的 **“下载 Excel 模板”** 按钮。在下载的表格中填入您的数据（日期、描述、金额、类别），保存文件。
   - **方式 B：导入现有 Excel**  
     点击 **“导入 Excel 数据”** 按钮，选择您整理好的 `.xlsx` 或 `.csv` 文件。系统会自动读取并转换为 AI 可读格式。
   - **方式 C：直接粘贴**  
     直接在文本框中粘贴 CSV 格式的数据（例如：`2024-01-01, 办公采购, -500, 运营`）。
3. **选择审计重点（场景）**：
   - **综合概览**：提供全面的财务状况分析。
   - **反欺诈检测**：重点寻找整数金额异常、周末交易、重复付款等欺诈迹象。
   - **税务合规**：检查不可抵扣费用、增值税合规性等。
   - **内控审计**：关注越权审批、拆单报销等内控风险。
4. **运行审计**：点击 **“运行审计”** 按钮。
5. **查看报告**：
   - **执行摘要**：AI 生成的总体评价。
   - **关键指标**：收入、支出统计及环比变化。
   - **风险提示**：按严重程度（高/中/低）列出具体风险点及整改建议。
   - **可视化图表**：收支分类柱状图。
6. **导出/打印**：点击右上角的 **“导出报告”** 可调用浏览器打印功能保存为 PDF。

### 2.2 凭证文档扫描

此模块用于通过 OCR 和视觉模型分析发票、收据、合同或 PDF 文档。

**操作步骤：**

1. **进入模块**：点击仪表盘的 **“文档扫描”** 卡片。
2. **上传文件**：
   - **上传图片/PDF**：点击上传区域，选择本地文件。
   - **使用相机**：点击 **“使用相机”**（仅限移动端或带摄像头的设备），对准单据拍照。
3. **选择凭证类型**：
   - **通用收据**：提取金额、日期、商户名。
   - **防伪/篡改检测**：AI 会仔细检查字体不一致、修图痕迹等伪造风险。
   - **增值税/税务发票**：提取税号、税额，验证合规性。
   - **合同审查**：检查签名、关键条款缺失等。
4. **立即分析**：点击按钮，AI 将读取图像内容并生成分析报告。

### 2.3 AI 智能咨询助手

这是一个专门经过财务知识微调的聊天机器人。

**使用场景：**
- 询问会计准则（如：“IFRS 16 租赁准则的主要变化是什么？”）。
- 咨询税务政策。
- 对刚才的分析结果进行追问。

**注意**：您可以随时在对话框中切换 AI 模型提供商（如从 Gemini 切换到 GPT-4o），以获得不同的回答视角。

## 3. 设置与配置

### 切换语言
点击顶部导航栏右上角的 **地球图标 (EN/中文)**，可在英文和中文界面之间一键切换。

### 切换 AI 模型
在顶部导航栏的 **“模型 (Model)”** 下拉框中，您可以选择底层使用的 AI 引擎：
- **Gemini (默认)**：Google 最新模型，速度快，免费额度高，综合能力强。
- **DeepSeek**：在中文语境和推理任务上表现优异。
- **GPT-4o**：OpenAI 旗舰模型，逻辑推理能力最强。
- **Qwen (通义千问)**：阿里云模型，适合处理国内商业语境。

*(注：部分模型可能需要管理员在后台配置相应的 API Key 才能使用)*

## 4. 常见问题与隐私

**Q: 上传的财务数据安全吗？**
A: AuditAI 仅将您的数据用于当次 AI 推理，不会用于模型训练。建议在上传前**对敏感信息（如真实姓名、银行卡号）进行脱敏处理**（例如用 "客户A" 代替真实名称）。

**Q: Excel 导入失败怎么办？**
A: 请确保 Excel 文件没有加密，且数据主要集中在第一个工作表（Sheet1）。推荐使用系统提供的“下载模板”进行填写以保证格式兼容。

**Q: 摄像头无法打开？**
A: 请确保浏览器已获得摄像头权限。在 iOS 设备上，请使用 Safari 浏览器；在 Android 设备上，请使用 Chrome。如果是本地部署，请确保使用 `localhost` 或 `https` 协议访问。
